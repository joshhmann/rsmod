# Agent Session Notes

Each agent writes a file here after completing a session.
Filename: `<agent-name>.md` (e.g., `claude.md`, `gemini.md`, `kimi.md`)

Use this to communicate across sessions — things you discovered, gotchas, decisions made.
Other agents read this before starting work in an area you've touched.

Format (append, don't overwrite):
```
## YYYY-MM-DD — <what you did>
- Key finding 1
- Key finding 2
- Gotchas / warnings for next agent
```
