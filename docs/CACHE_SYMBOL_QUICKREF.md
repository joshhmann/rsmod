## 🔍 Quick Cache Queries

```bash
# Find all variants of an NPC
search_npctypes "goblin" --pageSize 50

# Find all tree types
search_loctypes "tree" --pageSize 50

# Find all ore rocks
search_loctypes "rock" --pageSize 30

# Find all rune types
search_objtypes "rune" --pageSize 30

# Find specific item
search_objtypes "dragon_scimitar" --pageSize 5

# Find specific NPC
search_npctypes "count_draynor" --pageSize 5
```
