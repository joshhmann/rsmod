DTX Integration Plan

Status: deferred until F2P content is complete.

Rationale
- RSMod already has a functional drop-table system for current F2P NPCs.
- DTX offers advanced roll styles (chained, exhaustive, meta, sequential) that are not yet required.

When To Revisit
- After all F2P content is implemented and validated.
- If we need accurate boss drops, tertiary rolls, or complex multi-roll tables.

Next Steps (When Reopened)
1. Evaluate gaps in RSMod drop-table logic against target NPCs.
2. Decide between extending `rsmod/api/drop-table` vs integrating DTX.
3. If integrating, define a migration path and authoring DSL for content scripts.
